## Packages
recharts | Dashboard analytics charts and data visualization
papaparse | CSV parsing for WMS data upload
@types/papaparse | TypeScript definitions for papaparse
date-fns | Human-readable date formatting
lucide-react | Icons for the UI
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility function to efficiently merge Tailwind CSS classes in JS without style conflicts

## Notes
- `recharts` is used for the Admin Dashboard to visualize conference metrics.
- `papaparse` is used in the Upload Data page to allow Admins to upload CSV files representing WMS Separation Details.
- The color palette is customized in `index.css` to match the requested Mustard Yellow and Blue branding.
- Routing is handled by `wouter`. Mobile-first design is heavily emphasized for the Driver (Equipe de Entrega) modules.
